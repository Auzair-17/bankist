## Installation

### Step 1: Clone the repository

```bash
git clone https://github.com/auzair-17/bankist.git
```

### Step 2: Go to project directory

```bash
cd forkify-js-recipe-app
```

### Step 3: Start development server

```bash
npm run dev
```
